DnD Python Game v0.5 (version # may need changing)

A Dungeons & Dragons Character Sheet Generator as well as Combat Tester.

# Windows:
1. Open the Windows folder where dnd.exe is located.
2. Double-click `dnd.exe`.

# macOS:
1. Open the Mac folder where dnd.app is located.
2. Double-click `dnd.app`.
3. If you see a security warning, right-click dnd.app → Open → Open.

- No Python installation is required.
- No additional packages are required.
- Saves and other files (if any) will be created in the same folder as the executable.

For updates or source code, visit: https://github.com/JJNara39/dndcode
